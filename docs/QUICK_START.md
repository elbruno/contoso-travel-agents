## ğŸ’¡ Quick Tips

### For First-Time Users

1. **Start Simple**
   - Use a sample prompt first
   - Get familiar with AI responses
   - Build up to detailed questions

2. **Be Specific**
   - Include budget, dates, interests
   - More details = better recommendations
   - Example: "5-day trip to Iceland, $2000 budget, love hiking"

3. **Ask Follow-Ups**
   - Dig deeper into suggestions
   - Clarify any questions
   - Refine your itinerary

### For Mobile Users

- **Carousel**: Swipe left/right on images
- **Chat**: Takes full screen when opened
- **Close**: Tap X button or back button

### For Keyboard Users

- **Tab**: Navigate through buttons
- **Enter**: Send chat message
- **Shift+Enter**: New line in chat
- **Esc**: Close chat (when focused)
